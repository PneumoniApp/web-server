Welcome to PneumoniApp

What is PneumoniApp

It is an auxiliary application in the diagnosis of pneumonia cases through the analysis of radiographs, applying artificial intelligence through a neural network model deployed on a client-server architecture using TCP communication between the server in charge of the analysis and the server responsible for displaying the results through a friendly interface for the end user.
